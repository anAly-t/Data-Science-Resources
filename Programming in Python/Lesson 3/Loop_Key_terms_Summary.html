Term Description Example
Break 🚨 This powerful command halts the loop immediately when a specified condition is met. Think of it as the emergency stop button for your loop! for i in range(10):
if i == 5:
break stops the loop when i reaches 5.
Continue ➡️ This handy tool skips the remaining code in the current iteration and jumps straight to the next one. It's like saying, "No thanks, let’s move on!" when certain criteria are encountered. for i in range(10):
if i % 2 == 0:
continue skips the even numbers and processes only the odd ones.
Pass ⏳ A quiet placeholder, pass does nothing at all. It’s your go-to for a syntactical placeholder for future code—perfect for times when the structure is required, but you're not ready to execute any operations just yet. if x > 0:
pass allows you to leave the conditional structure empty for now.
Iteration 🔄 The process of executing a set of instructions repeatedly. In a loop that runs 5 times, each execution of the loop is one iteration.
Condition ⚖️ A statement that evaluates to true or false, guiding the flow of execution. if x > 10: checks if x is greater than 10.
Control Flow 🛤️ The order in which individual statements, instructions, or function calls are executed in a program. Control flow changes with if, else, and loop structures.
Iteration Variable 🧮 A variable that takes on the value of each element in a sequence during a loop. for i in [1, 2, 3]:
# i takes values 1, 2, and 3
Sequence 🗂️ An ordered collection of items (e.g., lists, strings) that can be iterated over. [1, 2, 3] is a sequence of three integers.
Infinite Loop ♾️ A loop that never terminates due to a condition that always evaluates to true. while True:
# creates an infinite loop unless interrupted
Nested Loops 🌀 A loop inside another loop, allowing for complex iterations over multi-dimensional data. for i in range(3):
for j in range(2):
print(i, j) produces pairs of indices.
Loop Counter 🔢 A variable used to count the number of iterations in a loop. count = 0;
for i in range(5):
count += 1 counts up to 5.
List Comprehension 📝 A concise way to create lists based on existing lists, often involving loops. [x**2 for x in range(10)] creates a list of squares of numbers from 0 to 9.
Range Function 🎯 A built-in function in Python that generates a sequence of numbers, often used with for loops. range(5) generates the numbers 0 to 4.
Early Exit 🚪 Using break to terminate a loop before it naturally ends, often based on a specific condition. for i in range(10):
if i == 5:
break exits the loop at 5.
